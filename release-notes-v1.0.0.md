## PayVault SDK v1.0.0

First stable release of PayVault -- one unified API for African payment providers.

### Features
- Unified API for Paystack and Flutterwave
- Type-safe TypeScript SDK with full type definitions
- Embeddable checkout widget (JS + CSS)
- Smart retry with exponential backoff and jitter
- Unified webhook handling with signature verification
- 4 normalized transaction statuses across all providers
- Automatic amount conversion (major currency units)
- Custom provider support via Provider interface
- Structured error handling with provider context

### Install
```bash
npm install payvault-sdk
```
